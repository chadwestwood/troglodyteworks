# Troglodyte Works

Troglodyte Works is an AI-assisted game server hosting platform.

The first supported games are:

- ARK Survival Ascended
- ARK Survival Evolved
- Palworld
- Minecraft

The goal is to let users manage servers through a private portal and an AI assistant.

Customers should be able to:

- View server status
- Start, stop, and restart servers
- Change maps or worlds
- Manage mods and plugins
- Adjust server settings
- View logs
- Create and restore backups
- Ask the AI for help diagnosing problems

The AI assistant should not directly control the machine freely. It should use safe MCP tools that expose approved actions only.
