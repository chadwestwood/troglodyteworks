# ADR-0003: Linear Is the Current Work-Planning System

**Status:** Accepted  
**Date:** 2026-07-22

## Decision

Linear is the operational system of record for the one-person project's eight-week plan, completed work, upcoming work, bugs, and security remediation. Jira is not required. The TWE Engineering Tracker remains a future product concept rather than a current dependency.

## Consequences

- Documentation must not direct contributors to Jira as the active backlog.
- Material completed work and newly discovered defects should be reflected in Linear with visible user awareness.
- Building an internal tracker is optional future product work, not prerequisite infrastructure.
