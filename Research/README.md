# Research

Research documents inform architecture but are not proof of implementation.

Active repository research currently lives primarily under `docs/`:

- `docs/nitrado-api.md`
- `docs/discord-api.md`
- `docs/beacon-api.md`

Each implementation must still validate current official provider behavior, use minimum scopes, mock external calls in automated tests, and keep credentials out of source control, logs, screenshots, and documentation.
