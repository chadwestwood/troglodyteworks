Public Beta

Bring one real community into Trog.

The beta is focused on a dependable path from account creation to a connected game server and useful Discord answers.

Step 1

Create your identity

Create a TWE account, then connect Google and Discord from Account Settings. Discord identity is required for Trog authorization.

Create account

Step 2

Join a Community

Use an invitation from a Community owner. Membership and Discord server access remain separate approvals.

Open communities

Step 3

Connect hosting

Owners can connect a revocable Nitrado service token, discover an ASA server, and attach it without exposing the token to the browser again.

Step 4

Verify Trog

Install Trog in the approved Discord server, then try /server help, /server status, /server players, and /server mods.

Beta expectations

Safe, observable, and honest about limitations.

Read-only operations firstStatus, player, and mod reads are live. Restart execution remains deliberately disabled.
Least privilegeTrog answers only in connected guilds and approved channels using provider-approved capabilities.
Report failures clearlyIf a provider or runtime component is unavailable, Trog says so instead of inventing a result.
