Connect hosting

  1. Sign in to Nitrado, then open the hidden long-life token page.
  2. Enter a description such as Trog.
  3. Check only service. Do not select rootserver, user, ordering, or SSH permissions.
  4. Choose Create. Your token appears once as green text at the top of the page.
  5. Copy the entire green token immediately, return here, and paste it below.

Trog encrypts the token and never shows it again. You can revoke it from Nitrado at any time.

Disconnecting removes the stored credential and local bindings. Revoke the long-life token separately in Nitrado to invalidate it at the provider.
