# About This Blueprint

The TWE Blueprint is the guiding document for every decision made within Troglodyte Works.

It is not a technical specification.

It is not a business plan.

It is not a marketing document.

It is the constitution of the Troglodyte Works Experience (TWE).

Technology will change.

Programming languages will change.

Games will change.

Artificial intelligence will change.

Our purpose should not.

Whenever uncertainty exists, this document should be the first place we look before making a decision.

If a proposed feature aligns with the ideas contained here, it belongs.

If it does not, it should be reconsidered.
